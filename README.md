Ringkasan Aktivitas Praktikum

Praktikum ini dilaksanakan untuk memahami penggunaan sistem operasi Linux, khususnya Kali Linux, dalam pengelolaan file, layanan sistem, dan konfigurasi keamanan dasar. Kegiatan diawali dengan pembuatan struktur direktori kerja sebagai media penyimpanan hasil praktikum. Selanjutnya dilakukan pembuatan dan pengeditan beberapa file teks menggunakan editor berbasis terminal, kemudian diverifikasi menggunakan perintah standar Linux untuk memastikan isi file sesuai dengan ketentuan yang diberikan.

Tahapan berikutnya mencakup operasi manajemen file, meliputi penyalinan, penggantian nama, pemindahan file, serta pengaturan hak akses file. Pengaturan permission dilakukan untuk memahami mekanisme kontrol akses pada sistem Linux. Selain itu, dilakukan pengolahan isi file menggunakan berbagai perintah seperti pencarian teks, pemotongan data, pengurutan, penghapusan data duplikat, serta penggunaan pipeline untuk mengombinasikan beberapa perintah dalam satu alur eksekusi.

Praktikum juga melibatkan pembaruan sistem operasi dan instalasi beberapa tools keamanan jaringan sebagai persiapan analisis keamanan dasar. Informasi jaringan sistem diidentifikasi dan didokumentasikan untuk memahami konfigurasi jaringan yang digunakan. Selanjutnya dilakukan pemeriksaan port terbuka guna mengidentifikasi layanan jaringan yang aktif pada sistem.

Pada tahap lanjutan, layanan SSH dikonfigurasi dan dikelola menggunakan systemctl untuk mempelajari manajemen layanan sistem. File konfigurasi SSH dianalisis dengan mengidentifikasi parameter-parameter penting yang berkaitan dengan keamanan akses jarak jauh. Seluruh aktivitas praktikum didokumentasikan melalui pencatatan perintah dan pengambilan tangkapan layar sebagai bentuk verifikasi dan pelaporan hasil kegiatan.

Secara keseluruhan, praktikum ini bertujuan memberikan pemahaman dasar mengenai administrasi sistem Linux dan penerapan konsep keamanan sistem, yang menjadi landasan awal dalam bidang sistem operasi dan keamanan jaringan.
